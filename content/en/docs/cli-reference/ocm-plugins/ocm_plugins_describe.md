### Usage

```
ocm ocm plugins describe [<options>] {<plugin name>}
```

### Options

```
  -h, --help   help for describe
```

### Description


Describes provides comprehensive information about the capabilities of
a plugin.


### Examples

```

$ ocm describe plugins
$ ocm describe plugins demo

```
